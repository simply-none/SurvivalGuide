fieldset 元素通常用来对表单中的控制元素进行分组(也包括 label 元素)

 

Content categories Flow content, sectioning root, listed, form-associated element, palpable content.
Permitted content An optional <legend> element, followed by flow content.
Tag omission 不允许，开始标签和结束标签都不能省略。
Permitted parent elements Any element that accepts flow content.
DOM interface HTMLFieldSetElement

注意: 与几乎任何其他元素不同，WHATWG HTML Rendering规范建议 min-width: min-content 作为<fieldset>默认样式的一部分, 并且许多浏览器都实现了这种样式 (或者与其类似的东西).

属性Edit节

 fieldset  中包含所有全局属性

disabled HTML5
如果设置了这个 bool 值属性, 它的后代表单控制元素也会继承这个属性, 它的首个可选的 legend 元素除外, 例如, 禁止编辑. 该元素和它的子元素不会接受任何浏览器事件, 比如点击或者 focus 事件, 一般来说浏览器会将这样的元素展示位灰色.
form HTML5
规定fieldset所属的form表单. 这个属性的值与其所属的form的ID相同. 默认值是最近的那个form. 
      
name HTML5
控制元素分组的名称
The label for the field set is given by the first <legend> element that is a child of this field set.

示例Edit节

Example #1: 一个拥有 fieldset, legend 和 label 的表单节

<form action="test.php" method="post">
  <fieldset>
    <legend>Title</legend>
    <input type="radio" id="radio"> <label for="radio">Click me</label>
  </fieldset>
</form>
